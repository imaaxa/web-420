Explain what profiles are and how they are related to application semantics.

"A profile is defined not to alter the semantics of the resource representation itself, but to allow clients to learn about additional semantics (constraints, conventions, extensions) that are associated with the resource representation, in addition to those defined by the media type and possibly other mechanisms. (Wilde, 2013)"

There are three ways to link a document to its profile.

  • Use an IANA link relation link. (<link href="profile_url_here" rel="profile">)
  • Add a profile link to the content-type header. (application/collection+json;profile="profile_url_here")
  • Use the itemtype hypermedia control. (<div itemtype="profile_url_here">)

What are application semantics?

Application semantics is how we, as developers, can try to link things in the application to items in the real world.
